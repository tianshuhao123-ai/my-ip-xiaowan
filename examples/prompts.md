# 小晚配图示例

## 只做配图策略

```text
Use $xiaowan-illustrations 先不要生图。
请分析下面文章，输出 5 张左右的 shot list。每张写清：放在哪段后、主题、核心意思、结构类型、小晚在做什么、建议元素和短标注词。

<粘贴文章>
```

## 生成单张配图

```text
Use $xiaowan-illustrations 为“把一个很大的目标拆成今天能完成的一小步”生成一张正文配图。
小晚要把一张大纸卷拆成一张小卡片；保持白底、蜡笔涂鸦绘本风，画面不要标题。
```

## 生成一篇文章的系列图

```text
Use $xiaowan-illustrations 为下面文章生成 4 张系列正文配图。
每张独立生成；小晚必须保持同一角色；不要拼图，不要长文字。

<粘贴文章>
```

## 批量生成指令

```text
Use $xiaowan-illustrations 为以下 5 个主题分别生成 5 张独立配图：
1. 收集线索
2. 分类整理
3. 迈出第一步
4. 持续浇灌
5. 完成小闭环

每张都要使用小晚参考图，保持粉桶、服装与蜡笔绘本风一致；每张只讲一个动作。
```

## 局部改图

```text
Use $xiaowan-illustrations 编辑这张图：只把小晚手里的物品改成一张小纸条，其他角色、服装、粉桶、蜡笔质感、构图和白底都保持不变。
```
